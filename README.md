# Sign-up Form

A simple sign-up form built to practice HTML, CSS and JavaScript fundamentals, based on a static design reference.

This project focuses on layout structure, form styling, native HTML validation and a small amount of JavaScript for custom validation logic.

## Features

- Two-column layout with a full-height image sidebar
- Custom font loaded locally using @font-face
- Styled form inputs with focus and invalid states
- Password validation using native HTML constraints
- Password match validation implemented with JavaScript
- Accessible form structure with labels and semantic markup

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla)

## What I Practiced

- Page scaffolding and layout planning
- Flexbox and CSS Grid for layout
- Styling form controls and managing visual states
- Using :focus and :invalid pseudo-classes
- Leveraging HTML5 form validation
- Using setCustomValidity() for custom error messages
- Keeping structure, style and behavior separated

## Project Structure

- index.html
- style.css
- script.js
- assets/
  - fonts/
  - images/

## Notes

This project is not optimized for mobile devices. The focus was on desktop layout and form behavior.

No external libraries or frameworks were used.
